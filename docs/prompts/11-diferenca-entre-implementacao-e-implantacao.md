# Prompt: Diferença entre implementação e implantação

**Nota:** Prompt reconstruído a partir de uma interação de revisão terminológica.

**Prompt do Utilizador:**
No contexto de uma dissertação de Engenharia Informática sobre arquitetura de software, qual é a diferença entre os termos "implementação" e "implantação"? Explica quando devo usar cada termo em português académico europeu e dá exemplos aplicados a componentes de software, infraestrutura, servidores e disponibilização de serviços.
